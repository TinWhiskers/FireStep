Status Machine::step ( const Quad< StepDV > &  pulse)
virtual

The step() method is the "stepper inner loop" that creates the stepper pulse train to QUAD_ELEMENTS steppers. The pulse parameter must have a value of -1, 0, or 1 for each stepper.

Steppers can't be driven too quickly–they stall. Each axis has a usDelay field that specifies a minimum delay between pulses.

Implements firestep::QuadStepper.
